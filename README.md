# Tic-Tac-Toe
#### Tic-Tac-Toe is a desktop application based on popular paper-and-pencil Noughts and Crosses game.
#### There are two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a diagonal, horizontal, or vertical row is the winner. It is a solved game with a forced draw assuming best play from both players.

In the application there are two game options:
- single-player: game vs AI with difficulty adjustment (easy, hard)
- multiplayer: with second player on the same computer
